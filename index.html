First Conditional – British English Lesson

The first conditional describes real and possible future situations. We form it with:

  • If + present simple, will + infinitive
  • or the reverse order: Will + infinitive … if + present simple

Examples:

  • If it rains, we will stay at home.
  • We will miss the bus if we don’t leave now.
